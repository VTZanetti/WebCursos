INSTRUCTIONS FOR RUNNING WEBCURSO APPLICATION

There are several ways to start the WebCurso application:

1. USING BATCH FILE (Windows Command Prompt):
   - Open Command Prompt (cmd)
   - Navigate to the scripts directory:
     cd c:\Users\vitor\OneDrive\Documentos\Dev\WebCursos\scripts
   - Run the batch file:
     start.bat

2. USING POWERSHELL SCRIPTS:
   - Open PowerShell
   - Navigate to the scripts directory:
     cd c:\Users\vitor\OneDrive\Documentos\Dev\WebCursos\scripts
   - Run one of these commands:
     - For both backend and frontend: .\start.ps1
     - For backend only: .\start-backend.ps1
     - For frontend only: .\start-frontend.ps1

3. USING PYTHON STARTUP SCRIPT:
   - Open Command Prompt or PowerShell
   - Navigate to the scripts directory:
     cd c:\Users\vitor\OneDrive\Documentos\Dev\WebCursos\scripts
   - Run:
     python start.py

NOTE: Do not run batch files (.bat) with Python. Batch files are executed directly in Command Prompt.
